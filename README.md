# 📌 Web Scraper
A Web Scraper for a Baseball Stats Page

---

## 🎯 Learning Objective
- Understanding how web scraping works
- Use a library

---

## 🛠️ Technologies Used
- Java version: 17
- jsoup

---

## 🚀 How to Run
### Prerequisites
- Installed Java (version 17 or higher)
- jsoup library

### Steps
1. Clone this repository
2. Add jsoup library
3. Go to src folder: cd src
4. Compile and run the project:
  javac Main.java
  java Main
